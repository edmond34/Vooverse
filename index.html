What is Vooverse?

Vooverse is an upcoming outer space massively multiplayer online game designed as a one-of-a-kind safe, fun, and transparent virtual adventure. Players, or Voos, will be able to ...
  • Collect, trade, and sell Items in a protected virtual economy
  • Explore and build in Planets solo or together with friends
  • Create and join Guilds of many kinds
  • Deck out with Accessories, Faces, Costumes—the list goes on!

When will it be ready?

We do not know yet, but since you're so early, we would encourage noting the code SPACEPIRATE to redeem in the future for a special in-platform reward.

Will it be free to play?

Yes, Voos will enjoy core features at no cost. We will offer a membership for next-level benefits and increased access to the platform.

Can I support it?

Yes, we would recommend you to bookmark Vooverse.com and check back. You are welcome to share the news with friends.

Will there be early access?

Yes, the Pioneer Program gets first dibs on experiencing Vooverse. You can only join it if you've been invited.
